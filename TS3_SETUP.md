# Настройка TeamSpeak 3 для работы с TS6 OverLive

## Включение ClientQuery API

Для работы виджета с TeamSpeak 3 необходимо включить ClientQuery API.

### Шаг 1: Откройте настройки
1. Запустите TeamSpeak 3
2. Нажмите **Tools** → **Options** (или `Ctrl+S`)

### Шаг 2: Включите ClientQuery
1. Перейдите в раздел **Addons**
2. Найдите в списке **ClientQuery**
3. Поставьте галочку напротив **ClientQuery**
4. Нажмите **OK**

### Шаг 3: Перезапустите TeamSpeak 3
1. Полностью закройте TeamSpeak 3
2. Запустите снова

### Шаг 4: Проверка
После перезапуска ClientQuery API будет доступен на порту **25639**.

## Настройка виджета

1. Откройте настройки TS6 OverLive
2. Выберите версию **TeamSpeak 3**
3. Виджет автоматически подключится к ClientQuery API

## Устранение проблем

**Виджет не подключается:**
- Убедитесь что ClientQuery включен (галочка стоит)
- Перезапустите TeamSpeak 3
- Проверьте что TS3 запущен

**ClientQuery отсутствует в списке Addons:**
- Обновите TeamSpeak 3 до последней версии
- ClientQuery доступен в версиях 3.1.0 и выше
