# Быстрый старт

## Запуск в режиме разработки

```bash
export PATH="$HOME/.cargo/bin:$PATH"
npm run dev
```

## Сборка приложения

```bash
# macOS (универсальный бинарник для Intel и Apple Silicon)
npm run build:mac

# Windows
npm run build:win
```

## Основные функции

### Горячие клавиши
- `Cmd+Shift+T` (macOS) / `Ctrl+Shift+T` (Windows) - переключение сквозного режима

### Иконка в трее
- **Левый клик** - показать/скрыть окно
- **Меню трея**:
  - Показать - открыть окно
  - Сквозной режим - включить/выключить режим клика насквозь
  - DevTools - открыть инструменты разработчика (только в dev режиме)
  - Выход - закрыть приложение

### Настройки
Открываются автоматически при клике на иконку шестеренки в правом верхнем углу окна:
- **Прозрачность** - от 0% до 100%
- **Минимальный режим** - компактное отображение
- **Макс. пользователей** - ограничение количества отображаемых пользователей
- **Цвет акцента** - цвет для активных пользователей
- **Сквозной режим** - окно не реагирует на клики мыши

## Первый запуск

1. Запустите TeamSpeak 6
2. Запустите TS6 OverLive
3. При первом подключении TeamSpeak попросит разрешение - **обязательно установите галочку "Запомнить"**
4. Окно оверлея появится в правом верхнем углу экрана
5. Перетащите окно в нужное место - позиция сохранится автоматически

## Устранение проблем

### Иконка не появляется в трее
- Убедитесь, что приложение запущено
- На macOS иконка должна быть видна всегда (приложение работает как аксессуар)
- Проверьте системные настройки трея

### Не подключается к TeamSpeak
- Убедитесь, что TeamSpeak 6 запущен
- Проверьте, что Remote App API включен в настройках TeamSpeak
- Порт по умолчанию: 5899

### Окно не видно
- Проверьте, что окно не скрыто за другими окнами
- Кликните на иконку в трее для показа окна
- Попробуйте изменить прозрачность в настройках
