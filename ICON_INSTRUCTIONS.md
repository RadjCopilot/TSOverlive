# Инструкция по созданию иконок

## Шаг 1: Подготовка исходной иконки

1. Создайте PNG файл размером **1024x1024** пикселей
2. **Важно**: Фон должен быть **прозрачным** (не белым!)
3. Сохраните файл как `icon-source.png` в корне проекта

### Как убрать белый фон:

**Вариант 1: Онлайн (быстро)**
- Откройте https://www.remove.bg/
- Загрузите вашу иконку
- Скачайте результат с прозрачным фоном

**Вариант 2: Photoshop/GIMP**
- Откройте иконку
- Используйте Magic Wand Tool для выделения белого фона
- Нажмите Delete
- Сохраните как PNG с прозрачностью

**Вариант 3: Preview (macOS)**
- Откройте иконку в Preview
- Используйте Instant Alpha (⌘+⌥+A)
- Кликните на белый фон
- Удалите выделение

## Шаг 2: Генерация иконок

Запустите скрипт:

```bash
./create-icons.sh
```

Скрипт автоматически создаст:
- PNG иконки всех размеров (16x16 до 1024x1024)
- `.icns` файл для macOS
- `.ico` файл для Windows (требуется ImageMagick)

## Установка ImageMagick (для Windows .ico)

```bash
brew install imagemagick
```

Или создайте .ico вручную на https://convertio.co/png-ico/

## Результат

Все иконки будут в папке `src-tauri/icons/` с прозрачным фоном.
