# TS6 Overlay

Оверлей виджет для TeamSpeak 6 на macOS и Windows.

**Разработчик:** Radj

## Возможности

- Отображение активных пользователей в канале
- Автоматическая сортировка по последней активности
- Визуальное выделение говорящих пользователей
- Настраиваемая прозрачность
- Минималистичный режим
- Ограничение количества отображаемых пользователей
- Автоматические обновления
- Прозрачный оверлей поверх всех окон

## Установка

```bash
npm install
```

## Требования

- Node.js 18+
- TeamSpeak 6 Client
- macOS 10.13+ или Windows 10+

## Разработка

```bash
npm run dev
```

## Сборка

```bash
# Для macOS
npm run dist:mac

# Для Windows
npm run dist:win
```

## Интеграция с TeamSpeak 6

Приложение автоматически подключается к TeamSpeak 6 Remote App API через WebSocket (ws://127.0.0.1:5899).

При первом запуске TeamSpeak 6 попросит разрешение на подключение - обязательно установите галочку "Запомнить".
