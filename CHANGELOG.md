# Changelog

## [1.0.0] - 2025-11-30

### Исправлено
- ✅ **Иконка в трее**: Исправлена проблема с белым кругом вместо иконки
  - Создана монохромная иконка для macOS (автоматически адаптируется под тему)
  - Создана цветная иконка для Windows (синий микрофон)
  - Иконки встроены в бинарник через `include_bytes!`
  
- ✅ **Видимость трея**: Иконка теперь всегда видна, пока приложение запущено
  - Добавлен `ActivationPolicy::Accessory` для macOS
  - Приложение работает как аксессуар (не показывается в Dock)
  - Трей инициализируется при запуске приложения

### Добавлено
- Скрипт `create-tray-icons.sh` для пересоздания иконок трея
- Документация `TRAY_ICON_FIX.md` с подробным описанием исправлений
- Документация `QUICK_START.md` для быстрого старта
- NPM скрипты:
  - `npm run icons:tray` - пересоздать иконки трея
  - `npm run icons:app` - пересоздать иконки приложения

### Технические детали
- Платформо-специфичные иконки через условную компиляцию Rust
- `iconAsTemplate: true` для автоматической адаптации под тему macOS
- Иконки 32x32px в формате PNG
- macOS: 8-bit gray+alpha
- Windows: 16-bit RGBA
